# 프로젝트 기본 정보 (초보자용)

이 문서는 이 프로젝트가 어떤 언어/도구로 만들어졌고, 어디에 배포되는지 쉽게 설명합니다.

## 1) 사용 언어
- **JavaScript (자바스크립트)**  
  웹 브라우저에서 동작하는 코드를 만드는 언어입니다.

## 2) 프레임워크 / 뼈대
- **React**  
  화면을 만들기 위한 라이브러리입니다. 컴포넌트라는 작은 화면 조각들을 조합해서 앱을 만듭니다.
- **Vite**  
  React 프로젝트의 뼈대를 만들고, 개발 서버/빌드를 빠르게 해주는 도구입니다.  
  이 프로젝트는 `npm create vite` 방식으로 시작한 구조입니다.

## 3) 배포(호스팅) 위치
- **Firebase Hosting**  
  빌드된 웹 파일을 Firebase Hosting에 올려서 배포합니다.

## 4) 백엔드/서버
- **Firebase Functions**  
  서버 역할은 Firebase Functions가 담당합니다.  
  (예: AI로 책 내용을 생성하는 기능)

## 5) 데이터베이스
- **Firebase Firestore**  
  앱의 데이터는 Firestore에 저장됩니다.

## 6) 인증(로그인)
- **Firebase Authentication**  
  로그인/회원 기능은 Firebase Auth를 사용합니다.

---

### 참고: 배포 흐름(매우 간단히)
1. 코드를 수정한다
2. `main` 브랜치에 push한다
3. GitHub Actions가 자동으로 Firebase에 배포한다
