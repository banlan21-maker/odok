# 오독(odok) - AI 독서 앱

전체 문서는 **README.md** 를 참고하세요.

## 기술 스택
- React 19 + Vite + Tailwind CSS / Firebase / Capacitor(Android) / AdMob

## 주요 명령어
- `npm run dev` - 개발 서버 | `npm run build` - 빌드 | `npm run deploy` - Firebase 배포

## 주의사항
- 사용자는 비개발자입니다. 설명은 쉬운 말로 해주세요.
- 환경 변수(.env)나 비밀 키는 절대 커밋하지 마세요.
